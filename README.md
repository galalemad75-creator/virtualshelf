# VirtualShelf 🛒

Complete Supermarket Virtual Shopping System — 4 Apps, 1 Vision

## Apps

| App | Description | Theme |
|-----|-------------|-------|
| 📱 **Customer App** | Mobile shopping with barcode scanner, cart, QR confirmation | Dark Navy + Electric Blue |
| 🏭 **Warehouse Panel** | Order management for warehouse staff, pick lists, QR delivery | Dark + Orange |
| 🏪 **Store Manager** | Inventory dashboard, product management, stock history | Light + Green |
| 🖥️ **Admin Dashboard** | Executive KPIs, live orders, charts, activity feed | Dark + Purple/Gold |

## How to Run

Open any `index.html` file in a browser. No build step needed.

```bash
# Or serve locally
cd virtualshelf
python3 -m http.server 8080
# Open http://localhost:8080
```

## Tech

- Pure HTML/CSS/JS — no frameworks, no build tools
- Google Fonts (Inter, Roboto, Nunito)
- Material Symbols Rounded icons
- Responsive phone mockups for mobile apps
